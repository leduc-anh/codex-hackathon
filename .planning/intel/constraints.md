# Constraints

- 24-hour hackathon, 1-2 developers.
- Demo path first; defer everything not on P0.
- No extra dependencies beyond locked stack without human approval.
- TypeScript strict; no `any`; Zod validation at boundaries.
- Secrets via env only; no PII in logs.
- UI must support Vietnamese primary, English secondary, and no visible percentages for fit.
- Degraded states must continue the flow rather than crash.
